# mood-music-app
🎵 Real-time facial emotion detection app that recommends Tamil songs based on your mood using face-api.js and webcam
🎭 MoodTunes — Camera-Based Mood Detection Music Recommender

MoodTunes is a frontend web application that uses your device's 
webcam to detect your facial emotions in real-time and recommends 
Tamil songs that match your current mood.

Built with pure HTML, CSS, and JavaScript — no frameworks needed.

✨ Features:
- 📷 Live webcam feed with face detection
- 🧠 AI-powered emotion recognition (happy, sad, angry, neutral and more)
- 🎵 Tamil song recommendations based on detected mood
- 🔀 Random song suggestions — different playlist every time
- ▶️ YouTube links to listen instantly
- 🌙 Beautiful dark theme UI with animations
- 💻 Works completely in the browser — no server needed

🛠️ Tech Stack:
- HTML5, CSS3, Vanilla JavaScript
- face-api.js (TensorFlow.js based emotion detection)
- YouTube search integration
